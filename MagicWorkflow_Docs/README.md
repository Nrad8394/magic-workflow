# Magic Workflow Docs

MkDocs Material documentation for the Magic Workflow suite.

```bash
uv venv && uv pip install -e .
uv run mkdocs serve        # http://127.0.0.1:8000
# ...or: pip install -e . && mkdocs serve
```

Build / publish:

```bash
mkdocs build               # -> ./site
mkdocs gh-deploy           # GitHub Pages
```
